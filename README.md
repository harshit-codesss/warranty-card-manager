Warranty Card Manager

Project Overview
This project helps users digitize and manage product warranties. It calculates expiration dates and tracks how much time is left for a warranty claim.

 Features
- Add product details (Name, Purchase Date, Warranty Period).
- Dashboard showing "Days Remaining" for each item.
- Visual alerts for warranties expiring within 30 days.

Tech Stack
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Storage: LocalStorage (Current) / MERN Stack (Planned)
